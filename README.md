# AI Resume Analyzer

A full-stack AI-powered resume analysis platform with ATS scoring,
semantic job description matching, skill gap analysis, and AI-generated feedback.

## Tech Stack

| Layer      | Technology                                      |
|------------|-------------------------------------------------|
| Frontend   | React, Tailwind CSS                             |
| Backend    | Node.js, Express.js, JWT                        |
| ML Service | Python, FastAPI, Sentence Transformers, spaCy   |
| Database   | PostgreSQL                                      |
| AI         | Claude API (Anthropic)                          |
| Deployment | Docker, Docker Compose                          |
